What 'scope' is, and how it works in javascript

Scope refers to the reach of a variable within a code which is related to where is is declared. For example something like 'var globalVariable = 5;' will be a 'global' variable, and any other function or code can refer to it. But the variable in this function
'adder (a, b) { var c = a+b; return c; }'
'c' is a local variable, and its value can only be changed within the function. The 'c' can be passed out of the function with the 'return' and accessed that way. So
globalVariable = adder(5, 4)
would end up with the value of 'c' and as the adder() can access the globalVariable, add(globalVariable, 7); would work.

In the javascript language, describe all the use cases for each of:

  • parentheses () - e.g. '1. Execute a function var result = doSomething()'

    • Parentheses are used in functions to hold arguments and also to determine the order of mathmatical operations.

  • brackets []

    • Square brackets are used to hold the information contained in an array element and can be used to assign values or access elements of an array, ie thisArray[0] = 6; means the the value of the thisArray's first element is now '6'.

  • braces {}

    • Curly braces are used at the begining and end of a function and when called all the code within the brackets is executed.

  • single quotes '' and double quotes ""

    • Single or double quotes can be used to hold strings like "I am a cool dude". There is no prefered method so either can be used, but some JavaScript libaries use quotes differently and you should be consistent in you own code. You should also note using the same quote mark inside a string, i.e "I am a "cool" dude" will be read as "I am a", and likely lead to errors, so you should use the other type of quote if you want actual quote marks in your string i.e "I am a 'cool' dude"
